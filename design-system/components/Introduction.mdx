## O que é o Cortex DS

O Cortex DS é uma biblioteca de componentes React construída com:

- **Next.js 14+ App Router** — arquitetura de produção
- **Tailwind CSS v4** + CSS Variables — tokens integrados
- **Framer Motion** — animações premium e microinterações
- **shadcn/ui** (owned code) — componentes sob controle total
- **TanStack Table** — tabelas com sort, filter, seleção e paginação
- **Radix UI** — primitivas acessíveis (Dialog, Label, Tooltip)
- **Lucide React** — ícones consistentes

Todos os componentes são **dark mode first**, **WCAG AA compliant** e construídos sobre os tokens da marca Cortex.

|-------|-----|-----|

| `steel` | `#1E2533` | Bordas, divisores |
| `ash` | `#4A5568` | Texto desabilitado |
| `mist` | `#94A3B8` | Labels, captions |
| `white` | `#F7F9FC` | Texto principal |
| `signal` | `#4D71FA` | Accent — CTAs, links |
| `success` | `#22C55E` | Estado positivo |
| `warning` | `#F59E0B` | Estado de atenção |
| `error` | `#F43F5E` | Estado de erro |

## Estrutura Atômica

```
components/
├── atoms/          # Button, Input, Badge, Avatar, Spinner, Tag, Label, Tooltip
├── molecules/      # Card, Alert, Stat, FormField, Breadcrumb, Pagination
├── organisms/      # Modal, Sidebar, PageHeader, EmptyState, DataTable
└── templates/      # DashboardTemplate, CRMListTemplate, DetailTemplate, AuthTemplate
```
